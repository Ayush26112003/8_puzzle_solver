# 🧩 Intelligent 8-Puzzle Solver & Search Space Visualizer An interactive full-stack web application designed to demonstrate, analyze, and visualize classical Artificial Intelligence search strategies using an 8-puzzle sliding grid. This platform shifts pathfinding algorithms from abstract terminal logs into a tactile, visual analytics interface—making it an ideal tool for academic evaluations, presentations, and technical demonstrations. --- ## 🏫 Institutional Profile & Team * **Institution:** Techno Main SaltLake * **Department:** Computer Science & Engineering (CSE) * **Project Team Members:** * **Anuska Dey** (Roll No: 13030523008) * **Avinanda Guchait** (Roll No: 13030523016) * **Ayush Kumar Shaw** (Roll No: 13030523018) * **Tapasi Garai** (Roll No: 13030523058) --- ## 🚀 Core Features ### 1. Interactive Manipulation & Grid Workspace * **Dual Interaction Systems:** Physics-aligned manual block sliding alongside an advanced HTML5 native Drag-and-Drop API allowing users to physically set custom initial configurations. * **Intelligent Shuffle Engine:** Generates randomized board states by making reverse legal movements from the target goal state, ensuring 100% mathematical solvability (preventing the parity-check dead-ends of completely random permutations). ### 2. Micro-Step Playback Engine * **Dynamic Media Controls:** Features fully decoupled timeline playback control options (⏮ Step Back, ▶ Play / ⏸ Pause, Step Forward ⏭). * **Scrubbable Timeline Matrix:** Pre-computes and holds chronological states arrays, mapping out exact board layouts for any stage in a solution sequence with real-time positional updates (e.g., Step X of Y). ### 3. State Space Exploration Deep-Dive * **Discovered/Generated Node Space:** Clicking the **Nodes Expanded** metric card triggers a modal window rendering a comprehensive grid gallery showing the specific board permutations generated during the search cycle (capped at the initial 500 nodes to guarantee client-side DOM layout stability). * **Winning Sequence Matrix:** Clicking the **Path Length** metric card abstracts away the search tree overhead, mapping out a direct horizontal vector timeline showing the specific path of execution followed to hit the objective state. ### 4. Comprehensive Performance Analytics Panel * **Simultaneous Multi-Algorithmic Execution:** Runs the exact same layout profile across **A* (Manhattan Distance)**, **A* (Misplaced Tiles)**, and **Breadth-First Search (BFS)** concurrently. * **Logarithmic Metric Charting:** Renders comparative charts via Chart.js on a logarithmic Y-axis to illustrate exponential search domain differences (e.g., A* expanding ~30 nodes vs BFS expanding >15,000 nodes for the exact same board setup). * **Exportable Data Report Engine:** Features a browser-native URI-encoder to parse benchmark tables into a downloadable performance report .csv document for external statistical reporting. --- ## 🛠 Project Architecture & Structure The codebase is built matching industrial software patterns: separating templates, modular processing algorithms, backend routing APIs, and declarative styling assets.
